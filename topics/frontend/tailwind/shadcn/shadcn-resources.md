## Shadcn resources

[AwesomeShadcn UI](https://shadcn.batchtool.com/)
: The website "Awesome Shadcn UI" is a curated directory of resources for Shadcn/UI, a utility-driven design framework. It includes over 200 items across categories like UI components, templates, SaaS and landing page designs, and tools. It supports developers working with frameworks like Next.js, Tailwind CSS, and more, offering free and premium templates, boilerplates, and design systems tailored for web and app projects.
